# Base Element

#tech/language/html

You can set global default attributes to tags by adding a `base` element

```html
<head>
   <base target="_blank">
 </head>
```

E.g., putting `target="_blank"` in base will make all links open in new tabs.

Base shouldn't have a closing tag
